# Elowen Living - Luxury Real Estate Website

A complete replica of the Sotheby's International Realty website design, rebranded as **Elowen Living**.

## Pages Included

1. **index.html** - Homepage with hero search, featured properties, and magazine sections
2. **properties.html** - Property search & listings with interactive map
3. **sell-with-us.html** - Sell with Us page with marketing features and stats
4. **agents.html** - Worldwide agents directory with search

## Design Features

- **Exact Sotheby's replica**: Dark navy (#0a1a2e) color scheme, gold accents (#c9a96e)
- **Typography**: Georgia serif for headlines, Arial sans-serif for UI elements
- **Navigation**: Sticky top nav with utility bar, sub-navigation tabs
- **Image Placeholders**: All images use styled placeholder blocks with labels
- **Interactive Map**: Styled map with location markers on properties page
- **Agent Cards**: Grayscale-to-color hover effect on agent photos
- **Responsive**: Mobile-friendly with breakpoints at 1024px and 768px
- **Scroll Animations**: Intersection Observer reveal effects
- **Hover Effects**: Subtle transitions on buttons, links, and cards

## How to Use

1. Open `index.html` in any modern browser
2. Navigate between pages using the top navigation
3. All image placeholders are labeled for easy replacement
4. Replace placeholder divs with actual `<img>` tags when ready
