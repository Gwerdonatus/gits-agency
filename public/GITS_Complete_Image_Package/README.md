## HOW TO USE

1. Copy ALL root files (favicon.ico, apple-touch-icon.png, etc.) to your website's public root
2. Copy all subfolders (/social, /pwa, /apple-touch, etc.) to your public root
3. Paste head-meta-tags.html contents into <head> of every page
4. Update JSON-LD with your real social media handles
5. Submit sitemap to Google Search Console
